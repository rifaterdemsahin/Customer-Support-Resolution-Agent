🧠 Methodology

The process behind building this AI Agent animated demo — from storyboard to voice-over to the Lacanian awareness that the journey is the destination.

1 Storyboard

We start with the storyboard — the landing page that maps the entire narrative arc across three acts. Before any code runs, we define the structure at index.html with act cards, scene lists, and tool links. This is the canvas that holds everything together — a single-page view of the full journey before diving into any one act.

  • Three act cards side-by-side with descriptions and scene breakdowns
  • Tool grid linking external services (TTS, transcription, media manager)
  • Info bar for the combined 32s animation and documentation

2 Specs for Acts and Scenes

After the storyboard, we write detailed specifications for each act and its inner scenes. The master spec at docs/SPEC.md captures every element — timeline durations, subtitle text, audio clips, image assets, GSAP animations, and debug panel behavior. Each scene gets its own markdown file under scenes/ with frame-by-frame animation tables.

  • Act 1 — 3 scenes across 8s: Problem Card → Policy Gap → Error Outcome
  • Act 2 — 3 scenes across 16s: Policy Check → Decision → Success
  • Act 3 — 3 scenes across 8s: Comparison → Metrics → The Lesson
  • Per-scene timeline tables with GSAP events, element IDs, and subtitle sync

3 Debug and Iterate with the LLM

This is where the real learning happens. We iterate with a large language model, building each act file incrementally:

  • Act 1 — The Problem (red, #e94560): The naive agent guesses wrong. We build the failure scenario first — infographics, gray-area overlays, error flashes. Debug timings, subtitle reveals, and scene transitions until the red-tinted tragedy flows smoothly.
  • Act 2 — The Solution (green, #00cc66): The resilient agent detects the policy gap and escalates. SVG escalation paths animate, green success checkmarks appear. Iterate on the decision-to-escalation transition until it feels decisive and clear.
  • Act 3 — The Transformative Lesson (cyan, #00d4aa): Side-by-side comparison. Error rate drops from 100% to 0%. The final message: "Know what you don't know." Each iteration sharpens the visual metaphor.

4 Voice-Overs via Record and Transcribe

Narration audio is the backbone of the animated demo. We use Record and Transcribe to capture and transcribe voice memos and draft narration scripts. Final narrations are generated via edge-tts using the en-US-GuyNeural voice (charismatic, +5% rate, -2Hz pitch) and placed in generated-audio/.

  • Record draft narration ideas as voice memos
  • Transcribe into text for script refinement
  • Feed refined scripts into Kokoro TTS for final MP3 generation
  • Sync audio playback with GSAP timeline in each act file

5 AI Image Generation

Scene backgrounds are generated through OpenRouter using google/gemini-3nanobanana-image. Each prompt describes a specific moment in the narrative — confused AI robot, policy gap warning, escalation handoff, success checkmark, split-screen comparison. Images are placed in generated-images/ and referenced as low-opacity background layers in each act file.

  • 8 scene-specific PNGs generated from text prompts
  • Used at reduced opacity (25–35%) as ambient stage backgrounds
  • Infographic overlays (workflow diagrams, gray-area uncertainty) complement the backgrounds

💡 The Lacanian Triad of Building

The fun and joy is building and being in the journey — not getting stuck on any single step.

🪞

The Imaginary

Visualizing the storyboard. Seeing the narrative arc in your mind before it exists. The 3-act structure is an image of order you project onto chaos.

🗣️

The Symbolic

Writing the specs, the script, the code. Language structures reality — every GSAP timeline, every CSS rule, every scene file is an act of symbolic creation.

🔥

The Real

The animation running. The feeling when all three acts loop seamlessly. The gap between what you planned and what actually renders — that's where learning lives.

Built iteratively. Debugged relentlessly. Deployed continuously. ✦
